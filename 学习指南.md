# Excalidraw 代码仓库学习指南

## 1. 仓库概览

这个仓库是 Excalidraw 的 monorepo，包含核心库、应用和示例：

- `packages/excalidraw/`：主要 React 组件库，发布为 `@excalidraw/excalidraw`
- `excalidraw-app/`：完整的应用代码，包含 excalidraw.com 的前端实现
- `packages/common/`、`packages/element/`、`packages/math/`、`packages/utils/`：核心共享包
- `examples/`：示例项目集成代码

## 2. 重点目录和职责

### `excalidraw-app/`
- 应用入口和 UI：`index.tsx`、`App.tsx`
- 全局配置：`vite.config.mts`
- 应用级组件：`components/`
- 数据、文件、协作逻辑：`data/`、`share/`

### `packages/excalidraw/`
- 编辑器核心组件与功能实现
- 负责对外 API、渲染和工具栏交互

### `packages/element/`
- 图形元素定义与操作逻辑
- 元素类型、变换、渲染相关功能

### `packages/math/`
- 几何计算、坐标变化、矩阵运算等数学基础

### `packages/common/` 和 `packages/utils/`
- 通用工具、类型定义、事件处理、平台适配等

## 3. 推荐学习顺序

1. 阅读 `README.md` 和 `CLAUDE.md`，了解项目定位与开发流程。
2. 关注 `package.json` 根脚本：
   - `yarn start`：启动应用开发服务器
   - `yarn build`：构建应用
   - `yarn test:typecheck`：TypeScript 类型检查
   - `yarn test:app`：运行 Vitest
3. 从 `excalidraw-app/src/index.tsx` 和 `App.tsx` 入手，理解应用层如何加载编辑器。
4. 再看 `packages/excalidraw/src/`，找到编辑器核心组件和导出的公共 API。
5. 学习 `packages/element/` 的元素数据结构和变换逻辑，这是编辑器绘图的核心部分。
6. 结合 `packages/math/` 了解坐标、变换、几何运算支持。

## 4. 关键学习路径

### 4.1 应用启动链路
- `excalidraw-app/index.tsx`：React 入口
- `App.tsx`：主应用组件
- `AppSidebar.tsx`、`AppMainMenu.tsx`：应用菜单与侧边栏
- `Excalidraw.tsx`：核心编辑器组件（位于 `packages/excalidraw/`）

### 4.2 编辑器核心
- `packages/excalidraw/`：编辑器组件、工具栏逻辑、交互事件
- `packages/element/`：元素定义、渲染、形状操作
- `packages/math/`：几何与数学工具
- `packages/common/`：类型、共享常量、事件系统

## 5. 开发与测试建议

- 推荐先运行 `yarn install` 并使用 Yarn workspaces 管理依赖
- 常用命令：
  - `yarn start`：本地开发
  - `yarn test:app`：单元测试
  - `yarn test:typecheck`：类型检查
  - `yarn fix`：格式化与 ESLint 修复
- 如果修改库包，先执行 `yarn build:packages`，以保证 `excalidraw-app` 能正确引用最新包

## 6. 贡献与阅读技巧

- 先看 `excalidraw-app/` 的界面逻辑，再向 `packages/excalidraw/` 和 `packages/element/` 深挖。
- 搜索常见入口函数：`App.tsx`、`index.tsx`、`Excalidraw.tsx`、`renderScene()`、`onPointerDown`。
- 关注 `packages/element/` 的类型定义，它决定了绘图数据结构和状态管理方式。
- 使用 `yarn test:update` 运行快照/测试，了解仓库对变更的验收方式。

## 7. 详细学习路线图

### A. 第1天：快速启动与结构理解
1. 运行 `yarn install` 安装依赖。
2. 运行 `yarn start`，打开本地开发环境。
3. 阅读 `README.md`、`CLAUDE.md` 和根目录 `package.json`。
4. 确认核心目录：
   - `excalidraw-app/`
   - `packages/excalidraw/`
   - `packages/element/`
   - `packages/math/`
5. 找到 `excalidraw-app/index.tsx`、`excalidraw-app/App.tsx`、`packages/excalidraw/src/Excalidraw.tsx`。

### B. 第2天：从应用到编辑器
1. 重点阅读 `excalidraw-app/App.tsx`，理解应用如何嵌入编辑器。
2. 追踪 `excalidraw-app/index.tsx` 到 `Excalidraw` 组件的渲染。 
3. 识别 `AppSidebar.tsx`、`AppMainMenu.tsx` 等应用 UI 与编辑器的连接点。
4. 探索 `excalidraw-app/data/` 中的本地存储、文件管理与协作逻辑。

### C. 第3天：深入编辑器核心组件
1. 进入 `packages/excalidraw/src/`，阅读核心组件的导出和入口。
2. 重点查看：
   - `Excalidraw.tsx`
   - `App.tsx`（若存在）
   - `scene/*` 或 `renderer/*` 相关文件
3. 查找编辑器的工具选择、交互事件分发、渲染循环和场景状态管理。

### D. 第4天：理解图形元素与几何计算
1. 进入 `packages/element/`，理解元素类型与属性定义。
2. 关注 `src/element.ts`、`src/shape.ts`、`src/transform.ts` 等文件。
3. 阅读 `packages/math/` 里的基础函数：
   - 坐标变换
   - 向量/矩阵运算
   - 几何求交/边界计算
4. 学习 `packages/common/` 的类型定义和通用工具。

### E. 第5天：实践与修改
1. 找一个小问题或改进点，例如：
   - 修改按钮文本
   - 调整初始化场景
   - 添加一个小 UI 控件
2. 使用 `yarn build:packages` 确认库包构建结果。
3. 运行 `yarn test:app` 和 `yarn test:typecheck` 验证改动是否安全。

### F. 第6天：扩展与贡献准备
1. 阅读 `dev-docs/` 了解文档站点和贡献流程。
2. 关注 `packages/excalidraw/` 的导出 API，思考插件和集成使用场景。
3. 查找仓库中的 `TODO` / `FIXME` 注释，定位可贡献的问题。
4. 参考 `yarn fix`、`yarn test:update` 规范更新代码。
